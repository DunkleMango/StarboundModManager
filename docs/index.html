A mod manager for the game Starbound

If you want to support the creators of Starbound, you can buy it here (or just visit their homepage)

The StarboundModManager enables the user to manage mods for the game Starbound with ease (works with Steam). No experience needed!

This application is in no way associated with the game Starbound and their creators Chucklefish!
It was created by me to simplify the tedious task of managing many different mods for the game.
